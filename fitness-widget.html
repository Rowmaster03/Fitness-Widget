Set up Apple Shortcuts automation
01 Open the Shortcuts app → tap + to create a new shortcut.
02 Add action: Get Health Sample → choose Steps, period Today.
03 Repeat for Active Energy Burned and Apple Exercise Time.
04 Add action: Open URLs and paste this page's URL with the values appended:
?steps=STEPS&cal=CAL&min=MIN using the "Text" action to build the URL dynamically.
05 Add an Automation (Personal) to run this shortcut on a schedule — e.g. every hour or at morning/noon/evening.
06 When the page opens with URL params, it reads them automatically — no manual entry needed.
